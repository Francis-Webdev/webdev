README 

The calculator supports both light and dark themes, allowing users to switch between them for a comfortable viewing experience.

Features
Basic Operations: Perform addition, subtraction, multiplication, and division.
Clear and Delete: Clear the entire display or delete the last entered character.
Square Root Calculation: Calculate the square root of a number.
Percentage Calculation: Calculate percentages.
Error Handling: Displays error messages for invalid operations or negative square root calculations.
Responsive Design: The calculator is designed to be responsive and works well on various screen sizes.
Theme Toggle: Switch between light mode and dark mode for better visibility.
How to Use
Input Numbers and Operators:

Click on the number buttons (0-9) to input numbers.
Use the operator buttons (+, -, *, /) to perform arithmetic operations.
You can also use the . button to input decimal values.
Clear and Delete:

Click the C button to clear the entire display.
Click the DE button to delete the last character entered.
Calculate:

After entering your expression, click the = button to calculate the result.

Error Messages:

If an invalid operation is attempted (e.g., division by zero), an error message will be displayed below the calculator.
Theme Toggle:

Click the Dark Mode button to switch to dark mode. The button text will change to Light Mode, allowing you to switch back to light mode.

Technologies Used
